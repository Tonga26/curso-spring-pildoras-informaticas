# Prácticas de Spring MVC y Hibernate (Legacy)

Este repositorio contiene mi progreso y ejercicios siguiendo el curso de **Spring** de Píldoras Informáticas, enfocado en el dominio de su arquitectura más profunda.

**Objetivo:**
Comprender las bases y la arquitectura interna del framework (DispatcherServlet, ViewResolver, configuración XML y Data Binding) y el ciclo de vida de los Beans, así como la integración manual con bases de datos relacionales, para adquirir una base sólida antes de transicionar a configuraciones modernas con Spring Boot.

**Tecnologías:**
- Java 17 (Adaptado a entorno Java 8 / Tomcat 9)
- Spring MVC 5 (Controladores, ModelAttribute, InitBinder)
- Spring Validation y Hibernate Validator (Expresiones Regulares, Validaciones Personalizadas)
- Manejo de Strings externalizado (ResourceBundle y Properties)
- Hibernate 5.4.33
- JDBC y MySQL Connector/J 8.4
- JSP / JSTL
- Apache Tomcat 9.0
