# Prácticas de Spring MVC (Legacy)

Este repositorio contiene mi progreso y ejercicios siguiendo el curso de **Spring MVC** de Píldoras Informáticas.

**Objetivo:**
Comprender la arquitectura interna del framework (DispatcherServlet, ViewResolver, XML Config) y el ciclo de vida de los Beans antes de migrar a configuraciones modernas con Spring Boot.

**Tecnologías:**
- Java 17 (Adaptado a entorno Java 8/Tomcat 9)
- Spring MVC 5
- JSP / JSTL
- Apache Tomcat 9.0
